About This Project

This application demonstrates a full-stack implementation of a gesture-controlled virtual mouse. The frontend captures webcam video and streams it to a Python backend via WebSockets.

The Python server uses OpenCV and MediaPipe to perform real-time hand tracking, interprets gestures, and sends control commands back to the frontend. This allows for a seamless, interactive experience right in your browser.
